# PathTracingInC
My attempt at making a graphics engine. This time, I chose to use C.
<br>
Rendring runs on the CPU.
<br>
<br>

## Download
```bash
git clone https://github.com/Stachu1/PathTracingInC.git
```
<br>
<br>

## Run
```bash
make run
```
<br>
<br>

## Movement
AWSD to move and drag mouse to rotate.
